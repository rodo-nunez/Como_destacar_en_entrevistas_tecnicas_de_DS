# Cómo destacar en entrevistas técnicas de Data Science

Materiales de la charla presentada en **DuocUC** por Rodolfo Núñez.

## Contenido

```
├── charla_entrevistas_ds.qmd       # Guión completo de la charla (output: HTML)
├── slides_entrevistas_ds.qmd       # Deck de presentación (output: RevealJS)
├── images/
│   ├── Logo_DuocUC.svg.png
│   └── Logo_principalEncoders.png
└── README.md
```

## Cómo renderizar

Requiere tener [Quarto](https://quarto.org) instalado.

```bash
# Slides para proyectar
quarto render slides_entrevistas_ds.qmd

# Guión completo en HTML
quarto render charla_entrevistas_ds.qmd
```

## Sobre la charla

Charla de 15 minutos (+15 de Q&A) orientada a estudiantes de ciencia de datos, con foco en cómo diferenciarse en entrevistas técnicas. Cubre errores comunes, mentalidad de solucionador de problemas, uso de Git, y por qué importa entender las herramientas que uno usa.
